# Robot Siguelíneas – Correcciones y Mejoras

Este proyecto es un fork de un robot siguelíneas basado en Arduino Nano.
Mi contribución se centró en la corrección del mapeo de pines del puente H
y en mejoras de estabilidad del control de motores.
![WhatsApp Image 2026-03-17 at 8 35 45 PM](https://github.com/user-attachments/assets/476dbdea-7a8a-4f0d-b032-71fa52efbc51)
## Contribuciones realizadas
- Corrección de la asignación de pines del puente H para Arduino Nano.
- Ajuste del código para evitar conflictos entre pines digitales.
- Mejora en la respuesta de los motores durante el seguimiento de línea.

## Tecnologías utilizadas
- Arduino Nano

- C / C++
- Módulo QTR-8A (sensores reflectivos)
- Driver TB6612FNG (puente H)

## Resultado
Después de las correcciones, el robot respondió de forma estable y consistente
a las señales de los sensores, eliminando fallos de movimiento.
